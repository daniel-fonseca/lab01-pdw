Pergunta: Considere o cenário em que você está trabalhando em um projeto de site com sua dupla. Qual seria sua abordagem inicial para organizar as tarefas, escolher ferramentas e implementar boas práticas de versionamento e desenvolvimento?
Resposta: Minha abordagem inicial seria dividir as tarefas entre frontend e backend, garantindo alinhamento sobre os requisitos e prioridades do projeto. Para as ferramentas, utilizaria HTML, CSS e JavaScript no desenvolvimento do frontend, enquanto no backend adotaria Node.js e Express, utilizando PostgreSQL para o armazenamento e gerenciamento de dados. Quanto ao versionamento, configuraria um repositório Git compartilhado, utilizando branches específicas para desenvolvimento (uma para cada US, sendo estas definidas pela documentação do projeto já realizada anteriormente) e aplicando boas práticas, como commits frequentes e descritivos, code reviews por meio de pull requests e implementação de pipelines de integração contínua (CI/CD) para automação de testes, builds e deploys.
